# NexaGames

NexaGames adalah website katalog game berbasis web yang menampilkan data game secara live dari API, dilengkapi fitur pencarian, detail game, dan autentikasi Google.

## Teknologi yang Digunakan

- CodeIgniter 4
- FreeToGame API
- Google OAuth Login
- MySQL
